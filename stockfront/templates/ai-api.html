The API

API instructions

In very simple terms, the api allows acces to the data created by the trading bots. Currently you can view their trading history, watchlists and their active trades trough the api.

The base url is the same as the webste, followed by /api.v1/ that initiates the version, this is followed by the algo you want to view ex. /charlie/. After this you add the type of data that you want by /watchlist/ other possible options are tradehistory and active_trades. Lastly you'll need to add a limit argument (max is 20) /limit=10/ and your token as &token="Your token"

If you do not have a token yet you can create one from your profile. Your token can be found from your. Once you account has a token you can click the link below to view an example api query.

www.stockfront.appspot.com/api.v1/charlie/tradehistory/limit=10&token={{token}}
